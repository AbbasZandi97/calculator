# Calculator App

A simple console calculator written in C#.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input validation

## Concepts Practiced

- Classes
- Static methods
- Separation of concerns (SRP)
- Input validation
- Switch expressions

## Project Structure

- Program.cs - application entry point
- InputHandler.cs - handles user input and choice
- Validator.cs - validates inputs and choice to be made
- Calculator.cs - performs calculations
- OperationStrategy.cs - chooses the operation based on the choice
- MessagePrinter.cs - displays messages

## How to Run

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Run the application.
